# 运行项目
```

git clone (Clone or download 下的 url)

npm install

react-native run-android

```
# apk 编译说明
```
cd im_core_mobile
cd android

```
## bash 打包方式
```
./gradlew assembleDemoRelease

```

## cmd 打包方式
```
gradlew.bat assembleDemoRelease

复制 im_core_mobile/android/app/build/outputs/apk/app-demoRelease.apk 到手机安装

```
